## 一、TypeHandler 体系

### 1.1 TypeHandler 接口

```java
// 位置: type/TypeHandler.java
// 功能: Java类型与JDBC类型的双向转换

public interface TypeHandler<T> {
    // Java → JDBC: 设置参数
    void setParameter(PreparedStatement ps, int i, T parameter, JdbcType jdbcType) throws SQLException;
    
    // JDBC → Java: 读取结果 (按列名)
    T getResult(ResultSet rs, String columnName) throws SQLException;
    
    // JDBC → Java: 读取结果 (按索引)
    T getResult(ResultSet rs, int columnIndex) throws SQLException;
}
```

**作用**：解耦Java类型与JDBC类型的转换逻辑

### 1.2 BaseTypeHandler 抽象类

```java
// 位置: type/BaseTypeHandler.java
// 核心: 模板方法模式 - 处理null值

public abstract class BaseTypeHandler<T> implements TypeHandler<T> {
    protected Configuration configuration;
    
    // 模板方法: 处理null值
    @Override
    public void setParameter(PreparedStatement ps, int i, T parameter, JdbcType jdbcType) {
        if (parameter == null) {
            // null处理
            ps.setNull(i, jdbcType == null ? Types.NULL : jdbcType.TYPE_CODE);
        } else {
            // 非null: 子类实现
            setNonNullParameter(ps, i, parameter, jdbcType);
        }
    }
    
    // 模板方法: 错误增强
    @Override
    public T getResult(ResultSet rs, String columnName) {
        try {
            return getNullableResult(rs, columnName);
        } catch (SQLException e) {
            throw new SQLException("Error getting result for column: " + columnName, e);
        }
    }
    
    // 子类实现
    protected abstract void setNonNullParameter(PreparedStatement ps, int i, T parameter, JdbcType jdbcType);
    protected abstract T getNullableResult(ResultSet rs, String columnName);
}
```

### 2.1 Executor 接口

```java
// 位置: executor/Executor.java
// 功能: SQL执行的核心接口

public interface Executor {
    // 更新操作
    int update(MappedStatement ms, Object parameter);
    
    // 查询操作 (带BoundSql)
    <E> List<E> query(MappedStatement ms, Object parameter, RowBounds rowBounds, 
                     ResultHandler resultHandler, BoundSql boundSql);
    
    // 查询操作 (不带BoundSql)
    <E> List<E> query(MappedStatement ms, Object parameter, RowBounds rowBounds,

    // 事务操作

    void rollback(boolean required);
    void close(boolean forceRollback);
}
```

### 2.2 BaseExecutor 抽象类

```java
// 位置: executor/BaseExecutor.java
// 核心: 模板方法模式

public abstract class BaseExecutor implements Executor {
    protected Configuration configuration;
    protected Transaction transaction;
    protected boolean closed;
    
    // 模板方法: query
    @Override
    public <E> List<E> query(MappedStatement ms, Object parameter, 
                          RowBounds rowBounds, ResultHandler resultHandler) {
        // 1. 获取BoundSql (处理动态SQL)
        BoundSql boundSql = ms.getBoundSql(parameter);
        
        // 2. 调用子类实现
        return query(ms, parameter, rowBounds, resultHandler, boundSql);
    }
    
    // 模板方法: query (带BoundSql)
    @Override
    public <E> List<E> query(MappedStatement ms, Object parameter,
                          RowBounds rowBounds, ResultHandler resultHandler,
                          BoundSql boundSql) {
        if (closed) {
            throw new RuntimeException("Executor was closed.");
        }
        return doQuery(ms, parameter, rowBounds, resultHandler, boundSql);
    }
    
    // 抽象方法: 子类实现
    protected abstract <E> List<E> doQuery(MappedStatement ms, Object parameter,
                                     RowBounds rowBounds, ResultHandler resultHandler,
                                     BoundSql boundSql);
    
    // 抽象方法: 子类实现
    protected abstract int doUpdate(MappedStatement ms, Object parameter);
    
    // 事务方法
    @Override
    public void commit(boolean required) {
        if (!closed) {
            if (required) {
                transaction.commit();
            }
        }
    }
}
```

### 2.3 SimpleExecutor 实现

```java
// 位置: executor/SimpleExecutor.java
// 核心: 简单执行器，每次创建新的Statement

public class SimpleExecutor extends BaseExecutor {
    
    protected <E> List<E> doQuery(MappedStatement ms, Object parameter,
                                 RowBounds rowBounds, ResultHandler resultHandler,
                                 BoundSql boundSql) {
        // 1. 获取数据库连接
        Connection connection = transaction.getConnection();
        
        // 2. 创建语句处理器 (注意: 每条SQL都创建新的)
        StatementHandler handler = configuration.newStatementHandler(
            this, ms, parameter, rowBounds, resultHandler, boundSql);
        
        // 3. 准备语句
        Statement statement = handler.prepare(connection);
        
        // 4. 设置参数
        handler.parameterize(statement);
        
        // 5. 执行查询
        return handler.query(statement, resultHandler);
    }
}
```

## 四、ResultSetHandler 体系

### 4.1 DefaultResultSetHandler

```java
// 位置: executor/resultset/DefaultResultSetHandler.java

public class DefaultResultSetHandler implements ResultSetHandler {
    
    // 核心方法: 处理结果集
    @Override
    public List<Object> handleResultSets(Statement stmt) {
        List<Object> multipleResults = new ArrayList<>();
        
        int resultSetCount = 0;
        ResultSetWrapper rsw = new ResultSetWrapper(stmt.getResultSet(), config);
        
        // 遍历ResultMap
        List<ResultMap> resultMaps = mappedStatement.getResultMaps();
        while (rsw != null && resultMaps.size() > resultSetCount) {
            ResultMap resultMap = resultMaps.get(resultSetCount);
            // 处理行数据
            handleResultSet(rsw, resultMap, multipleResults, null);
            rsw = getNextResultSet(stmt);
            resultSetCount++;
        }
        
        return multipleResults.size() == 1 ? multipleResults.get(0) : multipleResults;
    }
    
    // 处理行数据
    private void handleResultSet(ResultSetWrapper rsw, ResultMap rm, 
                       List<Object> results, ResultMapping parentMapping) {
        ResultSet resultSet = rsw.getResultSet();
        DefaultResultContext context = new DefaultResultContext();
        
        // 遍历每一行
        while (context.getResultCount() < rowBounds.getLimit() && resultSet.next()) {
            // 获取一行值
            Object rowValue = getRowValue(rsw, rm);
            // 保存结果
            context.nextResultObject(rowValue);
            resultHandler.handleResult(context);
        }
    }
    
    // 获取一行的值
    private Object getRowValue(ResultSetWrapper rsw, ResultMap rm) {
        // 1. 创建对象
        Object resultObject = createResultObject(rsw, rm, null);
        
        if (resultObject != null) {
            // 2. 自动映射 (列名→属性名)
            applyAutomaticMappings(rsw, rm, metaObject, null);
            
            // 3. 显式映射 (resultMap配置)
            applyPropertyMappings(rsw, rm, metaObject, null);
        }
        
        return resultObject;
    }
    
    // 创建结果对象
    private Object createResultObject(ResultSetWrapper rsw, ResultMap rm, String prefix) {
        Class<?> resultType = rm.getType();
        
        // 基本类型
        if (typeHandlerRegistry.hasTypeHandler(resultType)) {
            return createPrimitiveResultObject(rsw, rm, prefix);
        }
        
        // Bean对象
        return objectFactory.create(resultType);
    }
    
    // 自动映射
    private boolean applyAutomaticMappings(ResultSetWrapper rsw, ResultMap rm, 
                                   MetaObject metaObject, String prefix) {
        // 找出未映射的列
        List<String> unmappedColumns = rsw.getUnmappedColumnNames(rm, prefix);
        
        for (String column : unmappedColumns) {
            // 查找对应属性
            String property = metaObject.findProperty(column);
            if (property != null && metaObject.hasSetter(property)) {
                // 获取TypeHandler
                TypeHandler<?> handler = rsw.getTypeHandler(
                    metaObject.getSetterType(property), column);
                
                // 获取值
                Object value = handler.getResult(resultSet, column);
                
                // 设置值
                if (value != null || !propertyType.isPrimitive()) {
                    metaObject.setValue(property, value);
                }
            }
        }
        
        return foundValues;
    }
    
    // 显式映射
    private boolean applyPropertyMappings(ResultSetWrapper rsw, ResultMap rm,
                                MetaObject metaObject, String prefix) {
        // 使用resultMap中配置的映射
        for (ResultMapping mapping : rm.getPropertyMappings()) {
            String column = mapping.getColumn();
            String property = mapping.getProperty();
            
            // 获取值
            TypeHandler<?> handler = mapping.getTypeHandler();
            Object value = handler.getResult(resultSet, column);
            
            // 设置值
            metaObject.setValue(property, value);
        }
        
        return foundValues;
    }
}
```

## 五、MapperProxy 体系

### 5.1 MapperProxy

```java
// 位置: binding/MapperProxy.java

public class MapperProxy<T> implements InvocationHandler, Serializable {
    private SqlSession sqlSession;
    private final Class<T> mapperInterface;
    private final Map<Method, MapperMethod> methodCache;
    
    @Override
    public Object invoke(Object proxy, Method method, Object[] args) {
        // Object方法直接调用
        if (Object.class.equals(method.getDeclaringClass())) {
            return method.invoke(this, args);
        }
        
        // 获取MapperMethod
        final MapperMethod mapperMethod = cachedMapperMethod(method);
        
        // 执行
        return mapperMethod.execute(sqlSession, args);
    }
    
    // 获取方法缓存
    private MapperMethod cachedMapperMethod(Method method) {
        MapperMethod mapperMethod = methodCache.get(method);
        if (mapperMethod == null) {
            // 创建并缓存
            mapperMethod = new MapperMethod(mapperInterface, method, 
                sqlSession.getConfiguration());
            methodCache.put(method, mapperMethod);
        }
        return mapperMethod;
    }
}
```

### 5.2 MapperMethod

```java
// 位置: binding/MapperMethod.java

public class MapperMethod {
    private final SqlCommand command;
    private final MethodSignature method;
    
    public MapperMethod(Class<?> mapperInterface, Method method, Configuration config) {
        this.command = new SqlCommand(config, mapperInterface, method);
        this.method = new MethodSignature(config, method);
    }
    
    // 执行方法
    public Object execute(SqlSession sqlSession, Object[] args) {
        // 参数转换
        Object param = method.convertArgsToSqlCommandParam(args);
        
        // 根据SQL类型执行
        switch (command.getType()) {
            case SELECT:
                if (method.returnsMany) {
                    return sqlSession.selectList(command.getName(), param);
                } else {
                    return sqlSession.selectOne(command.getName(), param);
                }
            case INSERT:
                return sqlSession.insert(command.getName(), param);
            case UPDATE:
                return sqlSession.update(command.getName(), param);
            case DELETE:
                return sqlSession.delete(command.getName(), param);
            default:
                throw new RuntimeException("Unknown execution method");
        }
    }
    
    // SqlCommand: SQL命令信息
    public static class SqlCommand {
        private final String name;      // MappedStatement ID
        private final SqlCommandType type;  // SQL类型
        
        public SqlCommand(Configuration config, Class<?> mapperInterface, Method method) {
            String statementName = mapperInterface.getName() + "." + method.getName();
            MappedStatement ms = config.getMappedStatement(statementName);
            name = ms.getId();
            type = ms.getSqlCommandType();
        }
    }
    
    // MethodSignature: 方法签名
    public static class MethodSignature {
        private final boolean returnsMany;
        private final Class<?> returnType;
        private final SortedMap<Integer, String> params;
        
        public MethodSignature(Configuration config, Method method) {
            this.returnType = method.getReturnType();
            this.returnsMany = isCollectionOrArray(returnType);
            this.params = getParams(method);
        }
        
        // 参数转换
        public Object convertArgsToSqlCommandParam(Object[] args) {
            if (params.size() == 0 || args == null) {
                return null;
            }
            if (params.size() == 1) {
                return args[params.firstKey()];
            }
            // 多参数 → Map
            Map<String, Object> param = new ParamMap<>();
            for (Map.Entry<Integer, String> entry : params.entrySet()) {
                param.put(String.valueOf(entry.getKey()), args[entry.getKey()]);
                param.put("param" + (entry.getKey() + 1), args[entry.getKey()]);
            }
            return param;
        }
    }
}
```

### 6.1 核心功能

```java
// 位置: session/Configuration.java

public class Configuration {
    // 环境配置
    protected Environment environment;
    
    // Mapper注册中心
    protected MapperRegistry mapperRegistry = new MapperRegistry(this);
    
    // 映射语句
    protected final Map<String, MappedStatement> mappedStatements = new HashMap<>();
    
    // 结果映射
    protected final Map<String, ResultMap> resultMaps = new HashMap<>();
    
    // 类型处理器
    protected final TypeHandlerRegistry typeHandlerRegistry = new TypeHandlerRegistry();
    
    // 插件链
    protected final InterceptorChain interceptorChain = new InterceptorChain();
    
    // 对象工厂
    protected ObjectFactory objectFactory = new DefaultObjectFactory();
    
    // 创建方法
    public Executor newExecutor(Transaction transaction) {
        return new SimpleExecutor(this, transaction);
    }
    
    public StatementHandler newStatementHandler(Executor executor, MappedStatement ms,
                                          Object parameter, RowBounds rowBounds,
                                          ResultHandler rh, BoundSql boundSql) {
        StatementHandler handler = new PreparedStatementHandler(
            executor, ms, parameter, rowBounds, rh, boundSql);
        
        // 插件包装
        handler = (StatementHandler) interceptorChain.pluginAll(handler);
        
        return handler;
    }
    
    public ResultSetHandler newResultSetHandler(Executor executor, MappedStatement ms,
                                          ResultHandler rh, RowBounds rb, BoundSql bs) {
        return new DefaultResultSetHandler(executor, ms, rh, rb, bs);
    }
    
    public void addMappedStatement(MappedStatement ms) {
        mappedStatements.put(ms.getId(), ms);
    }
    
    public MappedStatement getMappedStatement(String id) {
        return mappedStatements.get(id);
    }
}
```

## 七、关键设计模式

### 7.1 模板方法模式

```
BaseExecutor.query() [模板方法]
       │
       ├─→ getBoundSql()   [公共逻辑]
       ├─→ doQuery()      [子类实现]
       └─→ close()       [公共逻辑]

BaseTypeHandler.setParameter() [模板方法]
       │
       ├─→ if (parameter == null) 处理null
       └─→ setNonNullParameter() 子类实现
```

### 7.2 代理模式

```
MapperProxy → JDK动态代理
       └─→ invoke() 拦截方法调用

PooledConnection → 连接池代理
       └─→ close() 拦截，返还连接池
```

### 7.3 责任链模式

```
InterceptorChain.pluginAll()
       │
       ├─→ interceptor1.plugin()
       ├─→ interceptor2.plugin()
       └─→ interceptor3.plugin()

调用时:
       └─→ invocation.proceed()
              ├─→ handler1.intercept()
              ├─→ handler2.intercept()
              └─→ handler3.intercept() → 目标方法
```

### 7.4 策略模式

```
TypeHandler接口
       │
       ├─→ StringTypeHandler
       ├─→ IntegerTypeHandler
       ├─→ DateTypeHandler
       └─→ 自定义TypeHandler
```

### 7.5 工厂方法模式

```
SqlSessionFactoryBuilder.build()
       └─→ SqlSessionFactory

DataSourceFactory
       └─→ DataSource

MapperProxyFactory.newInstance()
       └─→ MapperProxy
```

- BaseTypeHandler处理null值，子类实现具体转换
