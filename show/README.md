## 🌐 访问方式

### 本地访问
直接在浏览器中打开 `index.html` 文件，或者使用 Python 启动本地服务器:

```bash
cd show
python -m http.server 8000
```

然后访问 http://localhost:8000

### GitHub Pages
文档已配置自动部署到 GitHub Pages，访问地址:
https://rongx563647.github.io/LightSSM/show/
